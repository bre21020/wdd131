Overview

Purpose

My purpose is to make a website designed to show off the projects I've done. They can be related to art I've drawn, websites I've coded, or other projects, whatever I feel like showing off

Audience

Anybody interested in seeing what I can do

Dynamic elements

There will be two main columns, one for art and one for html projects. Each column will be populated by a list of projects, each project being created with javascript objects and inserted in.
